# @nextclaw/ui

Official NextClaw Web UI package.

This package is consumed by `nextclaw` and also published for standalone integration/development use.

## Install

```bash
npm i @nextclaw/ui
```

## Development

```bash
pnpm -C packages/nextclaw-ui dev
pnpm -C packages/nextclaw-ui build
```
